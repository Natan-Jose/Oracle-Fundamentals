▶️ Instale o Oracle Linux 8 (Minimal Install)
▶️ Crie as seguintes partições:
	▫️ /
	▫️ /home
	▫️ /tmp
	▫️ /boot
	▫️ /u01
	▫️ /u02
	▫️ /u03
	▫️ /backup
▶️ Instale o Oracle 19c via Slient Mode 
▶️ Crie dois databases:
	▫️ teste (NON-CDB)
	▫️ cdbteste1 (CDB)
